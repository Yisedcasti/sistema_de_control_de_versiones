# Logro Descripción Logro Materia
1 Desarrollar habilidades avanzadas de comprensión lectora, análisis crítico, producción escrita y oral, y aplicación de conocimientos literarios en proyectos de investigación. Identifica y analiza ideas principales y secundarias en diversos textos literarios y no literarios.
Elabora ensayos argumentativos y resúmenes críticos con estructura clara y coherente.
Realiza exposiciones orales y participa en debates, defendiendo puntos de vista con argumentos sólidos.
Desarrolla proyectos de investigación literaria, utilizando diversas fuentes y aplicando diferentes perspectivas críticas.
Redacta monografías y artículos académicos, siguiendo normas de citación y estilo.		 Español
